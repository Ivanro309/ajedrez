## 📖 Descripción

Este proyecto es una implementación completa del clásico juego de Ajedrez desarrollada bajo el paradigma de la Programación Orientada a Objetos en **Java 21**, utilizando **JavaFX** para una interfaz de usuario fluida y moderna. El proyecto sigue el patrón de diseño arquitectónico **Modelo-Vista-Controlador (MVC)** para separar la lógica de negocio de la interfaz visual, garantizando un código limpio, mantenible y escalable.

## ✨ Características

- **Reglas Oficiales Completas**: Movimientos legales para todas las piezas, captura al paso, enroque (corto y largo) y coronación de peones.
- **Interfaz Gráfica Moderna**: Tablero interactivo con animaciones, resaltado de movimientos posibles y piezas con diseño nítido utilizando JavaFX.
- **Arquitectura MVC**: Separación estricta entre `modelo`, `vista` y `controlador`.
- **Validación de Código Calidad**: Uso de PMD para análisis estático de código, asegurando las mejores prácticas en el desarrollo Java.
- **Pruebas Unitarias**: Validaciones robustas de los movimientos y reglas con JUnit 5.

## 🛠️ Requisitos Previos

Para ejecutar y compilar este proyecto, necesitas:

- **Java Development Kit (JDK) 21**
- **Apache Maven** (opcional si usas el wrapper `mvnw` incluido)

## 🚀 Cómo ejecutar el proyecto

1. **Clonar el repositorio** (si aplica) o navegar a la carpeta del proyecto:
   ```bash
   cd ajedrez
   ```

2. **Compilar y ejecutar la aplicación** usando Maven:
   ```bash
   ./mvnw clean javafx:run
   ```

3. **Ejecutar pruebas unitarias** (opcional):
   ```bash
   ./mvnw test
   ```
